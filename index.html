3

增量同步

PyWebView 驱动

告别基于 Electron 的臃肿体积。后端 Python 核心逻辑 + 前端 React 高性能 UI，轻量且强大。支持 headless 无头模式静默运行。

课程别名 & 筛选

"高等数学(B) (001234)" 名字太长？设置别名自动重命名本地文件夹。支持精确到 Tab（如只下载“课件”不下载“作业”）的筛选控制。

隐私优先设计

你的账号密码仅保存在本地 config.ini 中，绝不上传云端。代码完全开源，逻辑透明可查。
